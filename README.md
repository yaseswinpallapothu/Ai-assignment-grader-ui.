# AI Assignment Grader

Live Demo:
http://assignment-ui-frontends.s3-website-us-east-1.amazonaws.com

## Overview
AI-powered Assignment Evaluation Platform built using React, Vite and Tailwind CSS.

## Features
- Upload assignments
- AI-based grading
- Assignment history
- Dark mode
- Responsive design
- Fast UI

## Tech Stack

Frontend:
- React
- Vite
- Tailwind CSS

Deployment:
- AWS S3
- GitHub

## Installation

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

# 🎓 AI Assignment Grader

> Intelligent Assignment Evaluation Platform built with React, Vite and Tailwind CSS.

🌐 **Live Demo:**  
http://assignment-ui-frontends.s3-website-us-east-1.amazonaws.com

---

## 📌 Overview

AI Assignment Grader is a modern web application that helps automate assignment evaluation using AI-assisted workflows.

The platform provides an intuitive interface for uploading assignments, reviewing results, and managing grading efficiently.

---

## ✨ Features

✅ Assignment Upload  
✅ AI-Based Evaluation  
✅ Saved Assignment History  
✅ Responsive UI  
✅ Dark Mode Support  
✅ Fast Rendering with Vite  
✅ Modern Dashboard Experience

## ⚙ Installation

Clone project:

```bash
git clone https://github.com/RevanthSettipalli/assignment-ui.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build production:

```bash
npm run build
```
